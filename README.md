# ShortcutVault

A clean, fast, modern **personal shortcuts & commands hub** – all your most-used keybindings and CLI commands in one place.

One single-file Progressive Web App (PWA) – works offline, installs on phone/desktop, perfect for GitHub Pages.

### What's inside (initially)

- Vim
- Tmux
- Google Cloud CLI (gcloud)
- Bash / Linux essentials
- Windows shortcuts
- macOS shortcuts
- Fully editable – add, edit, delete your own shortcuts (saved in browser localStorage)

### Features

- Lightning-fast fuzzy search (press `/` to focus)
- One-click copy
- Dark mode + responsive design
- Export / import your custom shortcuts
- Printable view
- Installable as app (PWA)


Enjoy! ⌨️
